# About Me Page - High School Coding Project

This is a simple "About Me" webpage template created for a high school coding course.

## What's Included

- **index.html** - The main HTML structure of the page
- **style.css** - All the styling to make the page look great

## How to Use

1. Open `index.html` in your web browser to view the page
2. Customize the content:
   - Replace `[Your Name]` with your actual name
   - Update the interests to match your own hobbies
   - Edit the coding journey section with your personal experience
   - Replace the placeholder image with your own photo
   - Add your social media links

## What Students Will Learn

- HTML structure (headings, paragraphs, sections, lists)
- CSS styling (colors, layouts, fonts)
- CSS Grid and Flexbox for responsive layouts
- How to add images to a webpage
- Creating navigation and links
- Basic responsive design for mobile devices

## Customization Ideas

- Change the color scheme in `style.css`
- Add more sections (favorite projects, awards, etc.)
- Include links to your actual projects
- Add animations or transitions
- Upload your own profile picture

## Tips for Students

- To change colors, look for color codes in `style.css` (like `#4A90E2`)
- To replace the profile image, use your own image file or a different placeholder
- Experiment with different fonts, spacing, and layouts
- Try viewing the page on different screen sizes to see the responsive design in action

Have fun customizing your page!
